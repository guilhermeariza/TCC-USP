# Projeto de Pesquisa: B-Trees vs LSM-Trees em SSDs NVMe

Este repositório contém os artefatos para o Trabalho de Conclusão de Curso sobre análise comparativa de desempenho entre B-Trees e LSM-Trees.

## Estrutura

*   `benchmarks/`: Contém todo o código fonte, scripts e configurações para execução dos testes.
    *   Consulte `benchmarks/README.md` para detalhes técnicos.
*   `ProjetoDePesquisa.pdf`: Documento original com os requisitos.

## Como Começar

1.  Vá para a pasta `benchmarks`.
2.  Siga as instruções no arquivo `walkthrough.md` (gerado na pasta `.gemini` ou disponível na documentação do assistente) ou leia o `benchmarks/README.md`.
